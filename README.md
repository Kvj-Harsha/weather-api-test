   ```bash

   cd ocean-weather-data-app
   ```

   ```bash
   npm install
   # or
   ```

   ```bash
   NEXT_PUBLIC_WEATHER_API_KEY=your_api_key_here
   ```

4. Start the development server:

   ```bash
   npm run dev
   # or
   yarn dev
   ```
